redis 最常见的场景就是分布式缓存，分布式锁的场景。它本身提供的数据结构在实现一些功能时要比关系型数据库方便很多，比如点赞，好友关系等功能，并且不用担心并发以及性能问题。这里记录下springboot整合jedis和redisson框架 ヽ(ˋ▽ˊ)ノ

redisson
