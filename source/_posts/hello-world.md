## 快速开始

### Create a new post

``` bash
$ hexo new "My New Post"
```

以上就是如何快速创建一篇新的博客文章。

### Run server

``` bash
$ hexo server
```

在本地运行服务，测试博客的效果。

### Generate static files

``` bash
$ hexo generate
```

生成静态页面。

### Deploy to remote sites

``` bash
$ hexo deploy
```

部署本地博客到远程服务器。这一步结束之后就可以愉快的访问到自己的博客了。
